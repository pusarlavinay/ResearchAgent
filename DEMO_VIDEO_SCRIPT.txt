================================================================================
TROUBLESHOOTING
================================================================================

IF UPLOAD FAILS:
- Check backend is running
- Verify file format (PDF, DOCX, TXT)
- Check file size (< 50MB)
- Restart backend if needed

IF QUERY FAILS:
- Ensure documents are uploaded
- Check documents are selected
- Verify backend connection
- Check console for errors

IF SLOW PERFORMANCE:
- Close other applications
- Check system resources
- Restart services if needed
- Use smaller documents for demo

================================================================================
FINAL TIPS
================================================================================

1. PRACTICE FIRST:
   - Do a complete run-through before recording
   - Time yourself (aim for 5-7 minutes)
   - Prepare your talking points

2. SPEAK CLEARLY:
   - Speak slowly and clearly
   - Pause between sections
   - Emphasize key features
   - Show enthusiasm!

3. SHOW, DON'T TELL:
   - Let the system demonstrate itself
   - Point to features as you mention them
   - Use mouse movements deliberately

4. BE PROFESSIONAL:
   - Dress professionally (if on camera)
   - Use professional language
   - Stay focused on features
   - Avoid filler words (um, uh, like)

5. HIGHLIGHT VALUE:
   - Emphasize time savings
   - Mention accuracy improvements
   - Show ROI potential
   - Compare to alternatives
